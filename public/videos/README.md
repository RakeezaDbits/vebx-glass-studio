# Hero background video (4K HD – theme)

Place your 4K HD video here for the **hero banner background** (dark / red accent / orbs theme):

- **Filename:** `hero-bg.mp4`
- **Path:** `public/videos/hero-bg.mp4`

The app uses this path by default. The old Lovable-only URL (`/__l5e/...`) does **not** work on local dev or most hosts unless you override it.

**Optional env (project root `.env`):** set `VITE_HERO_VIDEO_URL` to a full or absolute URL if your MP4 is hosted on a CDN instead of `public/videos/`.

If the file is missing or fails to load, a gradient + orbs fallback is shown. For best fit use a dark, abstract or tech-themed 4K video (e.g. orbs, particles, subtle motion).

Free 4K options: Pexels, Videezy, Pixabay (search: "abstract orbs", "particles", "dark tech").
